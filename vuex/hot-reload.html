热重载

Vuex 支持在开发中使用webpack的Hot Module Replacement API 热重载 mutations，modules ，actions 和 getters，你也可以在 Browserify 里使用 browserify-hmr 插件来实现同样的功能。

对于mutations 和 modules，你只需要简单地调用 store.hotUpdate():
